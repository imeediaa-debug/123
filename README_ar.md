
نظام مراجعات جوجل
تطبيق متكامل لإدارة وتتبع أداء مراجعات جوجل عبر عدة علامات تجارية وفروع وموظفين.

المميزات
تتبع المراجعات اليومية: تسجيل مراجعات فترتي الصباح والمساء

إدارة متعددة العلامات التجارية: دعم لعدة علامات تجارية مع شعارات مخصصة

إدارة الفروع والموظفين: تنظيم البيانات حسب الفروع والموظفين

تقارير متقدمة: رسوم بيانية وتحليلات مرئية

تصدير البيانات: تصدير إلى PDF وصور

دعم اللغة العربية: واجهة مستخدم كاملة باللغة العربية من اليمين لليسار

التقنيات المستخدمة
واجهة المستخدم: React + Vite + Tailwind CSS + Framer Motion

الخادم: Node.js + Express + MongoDB + Mongoose

المصادقة: رموز JWT

رفع الملفات: Multer لإدارة الشعارات

وثائق API
المصادقة
POST /api/auth/login - تسجيل دخول المستخدم

العلامات التجارية
GET /api/brands - عرض العلامات التجارية

POST /api/brands - إنشاء علامة تجارية (مع رفع شعار)

PUT /api/brands/:id - تحديث علامة تجارية

DELETE /api/brands/:id - حذف علامة تجارية

المدخلات اليومية
POST /api/daily - حفظ جماعي للمدخلات اليومية

GET /api/daily?date=YYYY-MM-DD - الحصول على المدخلات اليومية

التقارير
GET /api/reports/brand - تقارير أداء العلامات التجارية

GET /api/reports/employee - تقارير أداء الموظفين

ملاحظات الأمان

استخدام كلمات مرور قوية لمستخدمي المدير

إعداد CORS صحيح للنطاقات في بيئة الإنتاج

النشر في بيئة الإنتاج
تعيين متغيرات البيئة:

MONGO_URI: سلسلة اتصال MongoDB

JWT_SECRET: سر عشوائي قوي

NODE_ENV: production
